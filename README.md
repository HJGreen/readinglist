# ReadingList

A simple web application for tracking your reading history. Built with React.

## Data Structure

```json
{
  "books": {
    "byId": {
      1: {
        "title": "string",
        "author": "string",
        "date_started": "datetime",
        "date_finished": "datetime"
      }
    },
    "allIds": [1]
  }
}
```
